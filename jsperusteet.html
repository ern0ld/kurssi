JavaScript-perusteita

Muuttujat voidaan esitellä JavaScriptissä termillä ”let” tai ”var”. Toisin kuin esim. Javassa, jossa datan tyypitys on tarkkaa. Käytetään ainakin aluksi mieluummin var-termiä, näissä on hienoinen ero.

Uusi otsikkoelementti JavaScriptillä:

uusiHeader = document.createElement(’h1’); //uusi otsikkoelementti
uusiHeader.innerHTML = ”Tähän otsikossa lukeva teksti”;
document.body.appendChild(uusiHeader); //asettaa otsikon sivulle

Uusi painike luodaan JavaScriptillä seuraavasti:

var uusiBtn = document.createElement(’button’); //uusi Button-elementti
uusiBtn.innerHTML = ”tähän painikkeessa lukeva teksti”;
document.body.appendChild(uusiBtn); //asettaa painikkeen sivulle

Olemassaolevan elementin käsittely id:n perusteella

var headerElement = document.getElementById(’elementin id’); //Hakee html-dokumentista elementin annetun id:n perusteella Nyt kyseistä elementtiä voi muokata tilanteen mukaan esim. headerElement.hidden = true; //piilottaa otsikkoelementin

JavaScript-tiedoston liittäminen HTML-tiedostoon

Html-tiedostossa (lisätään yleensä loppuun) JavaScript-tiedoston sijainti osoitetaan:
script src=”tiedostonimi.js">

Funktion luominen JavaScriptillä:

function teeJotain(){
console.log(”funktiota kutsuttu”) //tulostaa konsoliin mikäli funktiota on kutsuttu
}

Eventien eli tapahtuminen seuranta (tässä tapauksessa Buttonin klikkaus)

munPainike.addEventListener("click", teeJotain)

Toinen vaihtoehto Buttonin klikkauksen hallinnalle:

munPainike.onclick = function(){
console.log("klikattu")
}
